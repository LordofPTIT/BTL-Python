Cài đặt VN Phishing Guard Pro

Kết nối API

URL của máy chủ dùng để kiểm tra và báo cáo (bắt buộc, phải là HTTPS). Ví dụ: https://my-guard-api.onrender.com/api

Cập nhật & Cache

Tần suất extension kiểm tra cập nhật danh sách chặn từ API (5 - 1440 phút, mặc định 60).

Whitelist (Quản lý thủ công - Chức năng đang phát triển)

Các mục này sẽ không bị kiểm tra. Chức năng đồng bộ với server đang được phát triển.
